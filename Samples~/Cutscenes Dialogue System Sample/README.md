# Cutscenes Dialogue System Sample

This sample demonstrates a cutscene graph that includes the optional
`Dialogue/Start Conversation` node.

Open `Scenes/CutscenesDialogueSystemSample.unity` and press Play in a project
where Dialogue System is installed.

The sample installer seeds a scene-authored graph that exercises:

- `Entry`
- `Log`
- `Start Conversation`
- `Wait`
- `Finish`
